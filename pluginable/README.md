# Total.js Pluginable – User Registration Example

This project demonstrates how to build a fully functional user registration form using Total.js v5, PostgreSQL, and jComponent inside a Total.js __Pluginable application__.
The example shows how frontend, backend, database, and custom plugins work together in a real-world scenario.

## What This Project Includes
- Custom Total.js plugin
- User listing using j-DataGrid
- Form for creating and updating users
- Backend logic for CRUD operations
- PostgreSQL database integration

The plugin contains both frontend and backend implementation, making it a complete modular example.

## Technologies Used
- Total.js v5 (total5)
- PostgreSQL
- querybuilderpg
- jComponent (Total.js UI library)

## How to Run the Application
### Install dependencies:

```bash
npm install total5 querybuilderpg
```

### Start the application:

```bash
node index.js
```

### The application runs on:

```bash
http://localhost:8003
```
